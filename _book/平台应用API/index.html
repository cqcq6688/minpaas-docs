授权接口

获取Token

基本信息

Path： /api/open/authorize/token

Method： POST

接口描述：

{
    "appId": "rd0z64BeH8vwth2RxtOe6eJs8IqKIJhD",
    "appSecret": "IYwCPZ7nZKksovx8ZYoxiVfRTtNdIYUT",
    "duration": 3600,
    "customFields": {
        "productAppCode": "aishop",
        "envCode": "aishopdev"
    }
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
durationnumber必须3600过期时间
customFieldsobject必须自定义属性
├─ productAppCodestring非必须产品code
├─ envCodestring非必须业务环境code
├─ tenantCodestring非必须租户code
├─ tenantAppCodestring必须租户应用code
├─ additionalFieldsobject非必须附加字段
appIdstring必须密钥应用id
appSecretstring必须密钥

返回数据

名称类型是否必须默认值备注其他信息
codenumber非必须状态码

枚举: 200,500

枚举备注: 200 正常 500错误

datanull非必须数据包
messagestring非必须消息

请求参数

tenantAppCodestring必须租户应用code
tenantNamestring必须租户名称
tenantCodestring必须租户code

返回数据

名称类型是否必须默认值备注其他信息
codestring必须
datastring必须
messagestring必须
